# Multi-Agent Skeleton

Minimal, implementation-free scaffold for:
- Deep Research: plan -> search -> report -> email
- SDR: generate and deliver professional outreach

## Entry Points
- Gradio UI: `src/sdr_agent/app.py`
- Deep Research flow: `ResearchManager().run(query)`
- SDR flow: `SDRManager().run(query)`

## Status
Skeleton only. No runtime logic included.
